SECTION 4 — PEGASUS MARKETPLACE (LOGIN REQUIRED)
====================================================================
This is the core marketplace where all users interact.

MAIN MODULES:
• Seller Submissions
• Wholesale Marketplace
• Dreamscaper Projects
• Investor Capital Marketplace
• Buyer Marketplace
• JV Opportunities
• Profile & Ranking System

-------------------------------------------------------------
A. WHOLESALER WORKFLOW
-------------------------------------------------------------
Wholesalers (internal + external) can:

1. Submit Deals
    Fields:
     - property address
     - ARV
     - asking price
     - repair estimate
     - photos/docs
     - assignment fee request
     - property status

   Save to table: wholesale_deals

2. See their active deals
3. Edit/update deals
4. Receive offers from dreamscapers

Internal Pegasus Wholesalers get:
- Trusted badge
- Higher ranking authority
- Better visibility in marketplace

-------------------------------------------------------------
B. DREAMSCAPER WORKFLOW
-------------------------------------------------------------
Dreamscapers (flippers) can:

1. Browse wholesale deals
2. Submit offers / JV requests
3. Post their own projects (if they seek capital)
4. Publish deal structure:
   - debt deal (interest rate, term)
   - equity deal (profit split)
   - total raise needed
   - minimum check size
   - timeline

5. When funded → move project stage to active
6. Auto-sync this deal to Pegasus HQ

Table: dreamscaper_projects, capital_raise_posts

-------------------------------------------------------------
C. INVESTOR WORKFLOW
-------------------------------------------------------------
Investors can:

1. Browse all active capital raises
2. Filter:
   - debt / equity
   - location
   - operator (dreamscaper)
   - raise type
   - risk score

3. View:
   - deal deck
   - operator reputation score
   - historical performance
   - legal docs

4. Commit capital:
   - amount
   - terms accepted
   - counteroffer (optional)

Save to table: investor_commitments

Investor sees a dashboard of:
- active commitments
- interest schedule
- updates provided by flipper
- completed deals

-------------------------------------------------------------
D. BUYER WORKFLOW
-------------------------------------------------------------
Buyers can browse:
1. Pegasus renovated flips (Dreamscaper-finished)
2. Wholesale assignments (if they want contracts)

Buyers can:
- Save favorites
- Submit offers
- Request showings
- Download disclosures

-------------------------------------------------------------
E. PROFILE & REPUTATION SYSTEM
-------------------------------------------------------------
Every user must have:

• Profile page with:
  - role
  - join date
  - number of deals done
  - verified badge (if internal Pegasus)
  - rating (1–5)
  - reviews
  - rank tier (Bronze → Silver → Gold → Pegasus)

• Activity feed
• Completed deal history
• Standing in the community

Tables:
user_profiles
user_reviews
user_badges
user_rank_history
