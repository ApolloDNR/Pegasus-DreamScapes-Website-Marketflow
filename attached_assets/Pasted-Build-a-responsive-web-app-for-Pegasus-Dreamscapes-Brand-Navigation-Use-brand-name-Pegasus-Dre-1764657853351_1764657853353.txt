Build a responsive web app for Pegasus Dreamscapes.

Brand & Navigation

Use brand name: Pegasus Dreamscapes

Main slogan for website hero:

“Where Designed Profits Are Crafted, and Communities Are Elevated.”

Top navigation (desktop + mobile):

Logo (click → Home)

Sell (page for homeowners / sellers to submit properties)

Buy (page for buyers to view:

“Wholesale Deals”

“Renovated / Move-In Ready Properties”)

Invest (landing page for investors)

Community (forum / discussion hub)

Portal Login (or “Dashboard” button) – leads to the main portal app

Core Concept

Create one unified portal dashboard with role-based access. User roles:

Investor

Wholesaler

Buyer

Dreamscaper (Staff/Admin)

A single user can technically have more than one role, but show different dashboard views based on selected role.

Data Models (high level)

Define these core objects:

User

Deal / Property Listing

Project (a specific flip or development)

Investment (ties Investor + Project)

Milestone (for each Project)

Announcement

Message / Conversation

Offer / Intent to Invest

1) Portal Dashboard (Global Layout)

After login, show:

A role switcher (tabs or dropdown): Investor, Wholesaler, Buyer, Dreamscaper (Staff)

A main dashboard with widgets depending on role:

Summary cards

Recent activity

Notifications

Messages

Include a left sidebar with:

Dashboard (home)

Deals & Projects

Messages

Community

Settings / Profile

Logout

2) Investor Portal

For users with Investor role, add:

Dashboard widgets:

Total capital deployed

Projects invested in (count)

Expected returns / preferred return %

Upcoming distributions or target payout dates

Tabs / sections:

My Investments

List of projects they’re in: project name, location, status, amount invested, projected ROI, timeline

Progress bar based on milestones

Milestones

Milestone timeline per project (Acquisition, Demo, Rough-in, Finishes, Listing, Under Contract, Sold)

Each milestone shows % complete and date

Upcoming Projects (Invest Now)

List of open capital-raising projects

For each project show:

Funding goal

Amount raised so far

Min / max investment

Structure (Equity / Debt / Hybrid)

Target timeline and returns

Button: “Request to Invest”

Request to Invest Flow

Form where investor chooses:

Project

Amount they want to invest

How they want to invest:

Default: Limited Partner

Optional: “Request General Partner role” (checkbox)

Preferred terms (notes / counter-offer)

Announcements

Central feed for updates from Pegasus or Dreamscapers:

New deals

Project updates

Distribution notices

3) Wholesaler Portal

For Wholesaler role:

Dashboard widgets:

Active deals posted

Deals under review by Pegasus

Deals under contract with cash buyers

Total assignment fees earned (or projected)

Tabs / sections:

Post a Deal

Form to create a deal listing:

Property address & basic details

ARV, asking price, estimated repairs

Photos / files

Deal type (Assignment, Double Close, JV, etc.)

Deadline / expiration

My Deals

List of all posted deals with statuses:

Under review by Pegasus

Shown to Investors / Buyers

In Negotiation

Under Contract

Closed

Matches & Offers

Show interested Investors / Buyers

Track offers and negotiations

JV & Assignment Flow

Ability for Pegasus (Dreamscaper/Admin) to:

Take the deal directly for Pegasus

Or match the deal with another buyer (JV)

System should record a finder’s fee or JV split for Pegasus

4) Buyer Portal

For Buyer role (cash buyers + people buying renovated homes):

Dashboard widgets:

Saved properties

Active offers

Upcoming showings / inspections

Tabs / sections:

Browse Deals

View:

Wholesale deals

Renovated / retail listings

Filters:

Location

Price range

Property type

Rehab level

Cashflow / cap rate (if rental)

My Activity

Saved deals

Submitted offers

In-contract deals

Offer / Interest Flow

Simple “I’m Interested” / “Submit Offer” button on each listing

Form to send:

Offer price

Proof of funds (upload)

Notes (terms, closing timeline, etc.)

5) Dreamscaper (Staff/Admin) Portal

For Dreamscapers (Pegasus staff):

Admin abilities:

Manage users and roles

Approve or reject deals posted by wholesalers

Feature / highlight deals

Approve investment requests

Configure deal terms (finder’s fee, equity splits, etc.)

Post internal projects to raise capital

Projects & Capital Raising:

Dreamscapers should be able to:

Create a Project

Property / project details

Scope of work

Timeline (start / end, milestones)

Budget & use of funds

Set Capital Raise

Funding goal

Min / max per investor

Structure (Equity or Debt)

Projected returns and distribution structure

Launch to Investors

Mark project as “Open for Investment”

Investors can see and request to invest

Track Capital & Progress

Track commitments vs goal

Mark milestones complete

Send announcements to investors

Matching Engine:

View a list of all wholesaler deals and all interested investors/buyers

Ability to manually match a wholesaler’s deal with:

A buyer

An investor group

Pegasus itself

Record finder’s fee or JV split for Pegasus on each match
